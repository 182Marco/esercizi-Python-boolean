# 🐍 Python Exercises Collection (Jupyter Notebooks)

## 📚 Overview

This repository contains a collection of Python exercises organized inside **Jupyter Notebooks (`.ipynb`)**.

A **Jupyter Notebook** is an interactive Python environment that allows you to combine:

* 🧠 Code execution
* 📝 Notes in Markdown
* 📊 Outputs and visualizations

It is widely used in **data science, AI engineering, and education** because it makes learning and experimenting with code very intuitive.
